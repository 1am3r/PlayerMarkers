PlayerMarkers by 1am3r

Overview

PlayerMarkers is a Bukkit plugin which outputs the Location of the online and offline Players to a JSON file, that can be used by a generated Map (like with Minecraft-Overviewer) to view the Positions of the Players on the Map.

See the Bukkit Dev page for more Information and Downloads of the compiled Plugin.
